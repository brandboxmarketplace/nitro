Getting Started

First install Docker, then run this command to install the Nitro CLI and create your machine:

bash <(curl -sLS http://installer.getnitro.sh)

To add a site to your machine, cd to your Craft project and run:

nitro add

Learn more at craftcms.com/docs/nitro/2.x.
